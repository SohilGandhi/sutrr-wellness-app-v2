# Sexual Wellness App: Information Architecture (Sitemap)

This sitemap defines the user journey and navigation structure, ensuring compliance with Indian regulations and focusing on emotional safety.

## 1. Onboarding Flow (The Entry)
*   **Splash Screen**: Minimalist branding.
*   **Age Verification**: DPDP-compliant "Year of Birth" entry.
*   **Privacy Manifesto**: Brief on data encryption and Indian server hosting.
*   **Granular Consent**: Toggle-based selection for data sharing.
*   **Identity Setup**: Gender-neutral options (Male, Female, Non-Binary, Prefer not to say).

## 2. Main Navigation (Bottom Bar)
*   **Home (Dashboard)**: "Focus on Today" - Wellness score, daily tip, quick log buttons.
*   **Track**: Deep-dive into libido, mood, energy, and cycle logs.
*   **Learn**: Incognito-ready library of sexual wellness education.
*   **Connect**: Expert consultation (Sexologists, Therapists).
*   **Profile/Safe Space**: Settings, Biometric lock, "Delete My Data" (DPDP).

## 3. Page-by-Page Wireframe Concepts

### A. The Dashboard (Home)
*   **Header**: "Good morning, [User]" (Generic name/Greeting).
*   **Wellness Ring**: A calming visual representation of their current health status.
*   **Safe Log**: A prominent button to "Log Your Day" (Mood/Energy).
*   **Daily Insight**: A snippet from a peer-reviewed article.

### B. The "Safe Space" (Profile/Settings)
*   **Camouflage Mode**: Settings to change the app icon and notification style.
*   **Biometric Toggle**: Enable FaceID/Fingerprint for entry.
*   **Data Export/Delete**: Clear buttons for Indian data compliance.

### C. Expert Connect (Consultation)
*   **ABHA Link**: Section to link Ayushman Bharat Health Account.
*   **Expert Profile Cards**: Bio, rating, and "Book Anonymous Session".
*   **Secure Video Chat**: UI for end-to-end encrypted consultations.

---

## 4. Visual Navigation Legend
*   **Primary Action**: Bottom-centered Floating Action Button (FAB).
*   **Secondary Actions**: Simple tab-based navigation.
*   **Emergency Exit**: Fixed "Quick Exit" button in the 'Learn' section to instantly hide the app.
